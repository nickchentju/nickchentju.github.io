    这里是每周分享的第二期，主要介绍本周内接触的优秀应用、文章以及读过的好书。本周介绍的是 Windows 和 iOS 通知同步应用 Dell Mobile Connect、纸质书、编辑创造软件和笔记组织形式的文章以及《以利为利》这本书 。

    应用

    1. Dell Mobile Connect，可以在 Windows 上收到 iphone 的通知。本来是 Dell 为自家电脑开发的手机互联软件，但是也可以通过修改注册表在其他品牌的笔记本上安装并使用。具体可以参考少数派上的这篇文章。使用下来，同步效果不错，基本上所有的系统和 app 通知都可以同步，还可以实现在电脑上收发短信以及收听和拨打电话。

    文章

    1. 纸质书何以延续至今，讲述了在纸质书日益式微的今天，有哪些地方是电子书替代不了的。
    2. 看到一篇介绍著名编剧、作家还在使用 MS-DOS 时代的编辑创作软件的文章（ 《沙丘》编剧、《权游》作者：MS-DOS 我能再用五十年！ ），讲述了《沙丘》编剧、马丁老爷子等一种大佬在科技产品层出不穷的今天，依然使用了几十年前的老旧设备和软件进行创作的故事，文中介绍的设备如Win XP 电脑、IBM Model M 键盘，软件如 Movie Master、WordStar 等显得非常有趣。
    3. 看到一篇讲笔记组织形式的文章（高效能職場筆記法 - 1-為什麼要改變一天一頁、隨手記錄的習慣？ ），分析了按照时间顺序随手记录和按照项目的发展进行记录两种方式的优缺点。作者认为，按照时间记录虽然比较简便，符合人的认知，就像记日记一样把事情记下来备忘，但是作为工作上的日志来说其实是不便于检索，而且无法起到提醒、联系的作用。而按照项目来记的好处是可以在项目的进展过程中一目了然的看到自己做了什么，或者是在接下来的时间节点上将要做什么，并且当项目完成后，就能形成一篇完整的项目记录，而不必从每天的记录中再去翻找复制。况且时间一过，之前的想法和思考过程很可能就忘记了，这时候回顾和整理其实是一件极费时费力的事情。

    每周一书

    以利为利

    最早是从兰小欢老师的《置身事内》中看到的推荐，讲述中国地方政府从改革开放以来和中央的财政关系。书中主要对中央和地方关于财权和事权分配方案的沿袭变化作了梳理，尤其是分税制改革后，中央占据了税收的主要地位。地方政府为了追求发展，只能寄希望于预算外收入，也就是土地财政的由来。

    读完这本书，能对我国改革开放以来的地方经济发展的大致脉络有所了解，也能体会到地方政府在中央拿走了大部分的税收收入后，工作的重心继而转向了能够贡献财政收入的土地开发。一方面，中央掌握了更多的收入后可以通过支付转移的方式支持中西部落后地区的公共基础设施建设，完成如三峡工程、南水北调等国家级大工程；另一方面，城市土地开发往往忽略了农村地区农民的利益，在越来越热的建设浪潮中，他们失去了土地，却无法获得足够多的回报，而迈向城市又将面临户籍制度加诸于身的限制，无法充分享受城市化所带来的福利。
